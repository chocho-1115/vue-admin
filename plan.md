# Vue Admin 项目分析与优化计划

## 当前已实现的功能

| 类别 | 功能 |
|------|------|
| **认证** | 登录/登出、Token管理、localStorage持久化、路由权限守卫 |
| **布局** | 侧边栏折叠/展开、移动端抽屉适配、面包屑、标签页视图(右键菜单) |
| **主题** | 深色/浅色切换、localStorage缓存 |
| **路由** | 嵌套路由(3层)、白名单、KeepAlive缓存、404兜底 |
| **HTTP** | Axios封装、统一错误处理、EventBus解耦 |
| **Mock** | MSW + Faker.js、业务状态码体系 |
| **图标** | SVG雪碧图 + Element Plus图标双方案 |
| **其他** | NProgress进度条、Markdown渲染、GitHub Actions CI/CD |

---

## 优化方向

### 1. 依赖精简（追求极致）

| 依赖 | 建议 |
|------|------|
| `nprogress` | 可自研一个20行的进度条替代 |
| `normalize.css` | 可内联精简版CSS Reset |
| `element-plus` | 如追求极致精简，可考虑纯CSS+手写组件（但会牺牲开发效率） |

### 2. 功能完善

| 功能 | 说明 |
|------|------|
| 按钮级权限 | 指令或组件控制元素可见性 |
| 动态路由 | 后端返回菜单/路由数据，前端动态注册 |
| 国际化(i18n) | 多语言支持 |
| 数据导出 | Excel/PDF导出 |
| 全屏功能 | 页面或组件全屏 |
| 快捷键 | 常用操作快捷键支持 |
| 页面水印 | 动态水印 |
| 骨架屏 | 页面加载占位 |

### 3. 性能优化

| 优化项 | 说明 |
|--------|------|
| 路由懒加载 | `() => import(...)` 按需加载页面 |
| 虚拟列表 | 大数据量表格场景 |
| 图片懒加载 | `loading="lazy"` 或 Intersection Observer |

### 4. 开发体验

| 改进 | 说明 |
|------|------|
| TypeScript | 增加类型安全（会增加一定复杂度） |
| 单元测试 | Vitest + Vue Test Utils |
| 更多示例 | 图表、富文本、文件上传等常见场景 |

- SVG优化 (svgo)
- 访问历史记录功能
